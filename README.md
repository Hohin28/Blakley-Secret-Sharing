# Blakley Secret Sharing

Simple implementation of Blakley Secret Sharing Scheme using C++ and web technologies.

## Files

* 2.cpp → core logic
* index.html → UI
* script.js → logic
* style.css → styling

## Run

g++ 2.cpp -o blakley
./blakley

Open index.html in browser
